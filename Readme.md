# How to run from local repository

1. Clone the repository
2. Navigate to **backend** folder ,Run `npm install` and create .env file and add enviromental variables :
```
DATABASE_URL=
JWT_SECRET=
```
3. run npm start
4. Navigate to front-end folder and Run `npm install` command in your terminal inside the root folder
5. run npm run dev

6. Backend server running at `http://localhost:4000/`
7. Frontend server running at `http://localhost:3000/`
