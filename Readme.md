# How to run from local repository

1. Clone the repository
2. Navigate to **backend** folder ,Run `npm install` and create .env file and add enviromental variables:
```
DATABASE_URL=
JWT_SECRET=
```
3. Navigate to front-end folder and Run `npm install` command in your terminal inside the root folder


7. Backend server running at `http://localhost:4000/`
8. Frontend server running at `http://localhost:3000/`
